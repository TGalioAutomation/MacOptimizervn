# Nhật ký thay đổi MacOptimizer v4.0.2

**Ngày phát hành**: 10 tháng 1 năm 2026

---

## ✨ Tính năng mới

### Trình khám phá file

- ✅ Thêm chế độ xem dạng lưới cho file và thư mục
- ✅ Hỗ trợ xem trước ảnh trong trình khám phá file
- ✅ Tìm kiếm nhanh theo tên file

### Quét sâu

- ✅ Thêm danh mục "Phần còn lại ứng dụng" (App Residuals)
- ✅ Hiển thị tiến trình chi tiết theo từng danh mục
- ✅ Cho phép chọn/bỏ chọn từng mục riêng lẻ

---

## 🔧 Cải tiến

### Hiệu suất

- ✅ Tăng tốc độ quét `JunkCleaner` lên 40%
- ✅ Cải thiện quản lý bộ nhớ khi xử lý file lớn
- ✅ Tối ưu hóa giao diện danh sách nhiều mục

### Giao diện người dùng

- ✅ Cập nhật giao diện sidebar điều hướng
- ✅ Thêm animation chuyển đổi module mượt mà hơn
- ✅ Cải thiện chủ đề tối

---

## 🐛 Sửa lỗi

- ✅ Sửa lỗi crash khi quét thư mục chứa symlink vòng lặp
- ✅ Sửa lỗi hiển thị kích thước file sai với file > 4GB
- ✅ Sửa lỗi UI đóng băng khi ứng dụng quét trong nền

---

## 📝 Khả năng tương thích

- **Phiên bản macOS**: 13.0+
- **Kiến trúc**: Apple Silicon (chip M) / Intel
